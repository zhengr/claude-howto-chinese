# 代码审查清单

## 安全检查清单
- [ ] 没有硬编码的凭证或密钥
- [ ] 对所有用户输入进行输入验证
- [ ] 防止 SQL 注入（参数化查询）
- [ ] 对状态变更操作进行 CSRF 防护
- [ ] 通过正确的转义防止 XSS
- [ ] 受保护端点上的认证检查
- [ ] 资源上的授权检查
- [ ] 安全的密码哈希算法（bcrypt、argon2）
- [ ] 日志中不包含敏感数据
- [ ] 强制使用 HTTPS

## 性能检查清单
- [ ] 没有 N+1 查询
- [ ] 适当使用索引
- [ ] 在有益处实施缓存
- [ ] 主线程上没有阻塞操作
- [ ] 正确使用 async/await
- [ ] 大数据集进行了分页
- [ ] 数据库连接使用了连接池
- [ ] 正则表达式已优化
- [ ] 没有不必要的对象创建
- [ ] 防止内存泄漏

## 质量检查清单
- [ ] 函数少于 50 行
- [ ] 变量命名清晰
- [ ] 没有重复代码
- [ ] 正确的错误处理
- [ ] 注释解释 WHY 而非 WHAT
- [ ] 生产环境中没有 console.log
- [ ] 类型检查（TypeScript/JSDoc）
- [ ] 遵循 SOLID 原则
- [ ] 正确应用设计模式
- [ ] 代码自解释

## 测试检查清单
- [ ] 编写了单元测试
- [ ] 覆盖了边界情况
- [ ] 测试了错误场景
- [ ] 存在集成测试
- [ ] 覆盖率 > 80%
- [ ] 没有不稳定的测试
- [ ] 模拟了外部依赖
- [ ] 测试名称清晰
