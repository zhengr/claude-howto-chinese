# Claude How To - 设计系统

## 视觉识别

### 图标设计理念：指南针与代码括号

Claude How To 图标使用**指南针和 `>` 代码括号**来代表对代码的引导导航：

```
     N (绿色)
     ▲
     │
W ───>─── E     指南针 = 指引/方向
     │          > 括号 = 代码/终端/CLI
     ▼
     S (黑色)
```

这创造了：
- **视觉清晰度**：立即传达"代码导航指南"
- **象征意义**：指南针 = 找到方向；`>` = 代码/终端
- **可缩放性**：从 16px 到 512px 都适用
- **品牌一致性**：以最小配色方案匹配开发者工具美学

## 颜色系统

### 调色板

| 颜色 | 十六进制 | RGB | 用途 |
|-------|-----|-----|-------|
| 黑色（主色） | `#000000` | 0, 0, 0 | 主描边、文字、南针 |
| 白色（背景） | `#FFFFFF` | 255, 255, 255 | 亮色背景 |
| 灰色（次要色） | `#6B7280` | 107, 114, 128 | 次要刻度标记、次要文字 |
| 亮绿色（强调色） | `#22C55E` | 34, 197, 94 | 北针、中心圆点、强调线 |
| 近黑色（深色背景） | `#0A0A0A` | 10, 10, 10 | 暗色模式背景 |

### 对比度比率（WCAG）

- 白色上的黑色：**21:1** AAA
- 白色上的灰色：**4.6:1** AA
- 白色上的绿色：**3.2:1**（仅装饰，不用于文字）
- 深色背景上的白色：**19.5:1** AAA

### 强调色规则

**亮绿色 (#22C55E) 仅保留用于高亮：**
- 指南针北针
- 中心圆点
- 下划线/边框强调
- 切勿用作背景色
- 切勿用于正文

## 图标细节

### 指南针规格

指南针标记由以下几何元素构成：

```
元素               | 描边/填充      | 颜色
--------------------|----------------|------------------
外环               | 3px 描边       | 黑色 / 白色（暗色模式）
北刻度             | 2.5px 描边     | 黑色 / 白色（暗色模式）
其他基点刻度       | 2px 描边       | 灰色 / 白色 50%（暗色模式）
间基点刻度         | 1.5px 描边     | 灰色 / 白色 40%（暗色模式）
北针               | 填充多边形     | #22C55E（始终为绿色）
南针               | 填充多边形     | 黑色 / 白色（暗色模式）
> 括号             | 3px 描边       | 黑色 / 白色（暗色模式）
中心圆点           | 填充圆形       | #22C55E（始终为绿色）
```

### 尺寸进阶

```
16px  → 仅环 + 针 + V 形标记（最小化）
32px  → 添加基点刻度标记
64px  → 添加间基点刻度标记
128px → 完整细节，所有元素清晰
256px → 最大细节，加粗描边
```

## 自定义

### 更改强调色

要创建具有不同强调色的变体：

1. 将所有 `#22C55E` 的实例替换为你的强调色
2. 确保装饰元素的对比度保持在 3:1 以上
3. 保持黑/白/灰结构不变

### 缩放

```css
svg {
  width: 256px;
  height: 256px;
}
```

SVG 通过 viewBox 自动缩放——无需变换。
