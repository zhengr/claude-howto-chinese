# 课程测验 — 题库

每节课 10 道题目。每道题目包含：类别、题目文本、选项（3-4个）、正确答案、解释和复习章节。

---

## 第 01 课：斜杠命令

### Q1
- **类别**：概念型
- **问题**：Claude Code 中的四种斜杠命令类型是什么？
- **选项**：A) 内置命令、技能、插件命令、MCP 提示 | B) 内置命令、自定义命令、钩子命令、API 提示 | C) 系统命令、用户命令、插件命令、终端命令 | D) 核心命令、扩展命令、宏命令、脚本命令
- **正确答案**：A
- **解释**：Claude Code 包含内置命令（如 /help、/compact）、技能（SKILL.md 文件）、插件命令（命名空间 plugin-name:command）和 MCP 提示（/mcp__server__prompt）。
- **复习章节**：斜杠命令类型章节

### Q2
- **类别**：实操型
- **问题**：如何将所有用户提供的参数传递给技能？
- **选项**：A) 使用 `${args}` | B) 使用 `$ARGUMENTS` | C) 使用 `$@` | D) 使用 `$INPUT`
- **正确答案**：B
- **解释**：`$ARGUMENTS` 捕获命令名称后的所有文本。对于位置参数，使用 `$0`、`$1` 等。
- **复习章节**：参数处理章节

### Q3
- **类别**：概念型
- **问题**：当同名技能（.claude/skills/name/SKILL.md）和旧式命令（.claude/commands/name.md）同时存在时，哪个优先？
- **选项**：A) 旧式命令 | B) 技能 | C) 先创建的优先 | D) Claude 请用户选择
- **正确答案**：B
- **解释**：同名技能优先于旧式命令。技能系统取代了旧的命令系统。
- **复习章节**：技能优先级章节
