# Incident Response

構造化されたインシデントレスポンスのワークフロー：

1. インシデントレコードを作成
2. 重大度と影響範囲を評価
3. オンコールチームへ通知
4. 診断情報を収集
5. 対応活動を統率
6. 解決内容をドキュメント化
7. ポストモーテムを設定
