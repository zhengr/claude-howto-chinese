# Test Coverage Check

テストカバレッジと品質を解析する：

1. テストカバレッジの割合をチェック
2. テストされていないコードパスを特定
3. テスト品質をレビュー
4. 不足しているテストケースを提案
5. エッジケースが網羅されているかを検証
