# API Documentation Generator

完全な API ドキュメントを生成する：

1. API エンドポイントをスキャン
2. 関数シグネチャと JSDoc を抽出
3. モジュール / エンドポイントごとに整理
4. 例つきの Markdown を作成
5. リクエスト / レスポンスのスキーマを含める
6. エラーのドキュメントを追加
